# Pocket Pi Pet

A Tamagotchi-style virtual pet game for Raspberry Pi.

## Installation

These instructions are for a fresh installation on **Raspberry Pi OS Lite**.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Prc292/pipet.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd pipet
    ```
3.  **Install dependencies:**
    ```bash
    pip install -r Tamagotchi/requirements.txt
    ```

## How to Play

1.  Open a terminal.
2.  Run the game:
    ```bash
    python3 Tamagotchi/main.py
    ```
